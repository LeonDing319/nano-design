# Nano Design

Turn any image into living motion. Nano Design generates four distinct animated effects from a single upload: ASCII, Glitch Art, Liquid Art, and Marble Art.

Save your work in any of six formats. PNG, JPEG, SVG, and PDF capture a single frame, while GIF and MP4 preserve the full motion.

This is the core implementation of [nanodesign.app](https://nanodesign.app/).👋🏻
